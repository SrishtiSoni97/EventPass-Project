EventPass is a role-based web application built using Django that allows admins to manage events and users to browse events, purchase tickets, and receive QR-based event passes.

## Features
- Role-based access (Admin & User)
- Admin can create and manage events
- Users can browse events and purchase tickets
- Session-based authentication
- Stripe payment gateway integration with webhooks
- QR code generation for event passes
- Coupon management system

## Tech Stack
- Backend: Python, Django, Django ORM
- Frontend: HTML, CSS
- Database: MySQL / SQLite
- Payment: Stripe API
- Tools: VS Code, Postman, Git, GitHub
